# Society5.0

## ▼Society5.0とは
>・狩猟社会、農耕社会、工業社会、情報社会に続く、5番目の新たな社会を指し、日本が目指すべき未来社会の姿として政府で提唱<br>
>・サイバー空間（仮想空間）とフィジカル空間（現実空間）を高度に融合させたシステム（サイバーフィジカルシステム：CPS）により、経済発展と社会的課題の解決を両立する、人間中心の社会<br>
