# AWS Snow Family

## ▼AWS Snow Familyとは
>・物理ストレージデバイスを使用してAWS環境にペタバイト級の大量データを転送することができるサービス<br>
>・<br>
<br>

### ●種類
>・Snowcone<br>
>>・AWS Snowファミリーにおける最小サイズのデバイス（2.1kg/8TB）<br>

>・Snowball<br>
>>・データ移行及びエッジコンピューティングデバイスで２つのオプションがある<br>

>・Snowmobile<br>
>>・セミトレーラートラックが牽引する長さ14mの輸送コンテナ（1台/100PB）<br>
