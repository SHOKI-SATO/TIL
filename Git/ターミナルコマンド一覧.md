# ターミナルコマンド一覧

●cd　ファイル名<br>
-ディレクトリを移動します。<br>

●ls<br>
-ディレクトリの内容を表示します。 ls -a コマンドで、隠しファイルを含めたディレクトリ全内容を表示します。<br>
<br>
●mkdir　ディレクトリ名<br>
-ディレクトリを新規作成します。<br>
<br>
●rm　ファイル名<br>
-ファイルを削除します。<br>
<br>
●cp　ファイル名<br>
-ファイルをコピーします。<br>
<br>
●mv　ファイル名<br>
-ファイルの移動とファイル名の変更を行います。<br>
<br>
●cat　ファイル名<br>
-ファイルの中身を表示します。<br>
<br>
