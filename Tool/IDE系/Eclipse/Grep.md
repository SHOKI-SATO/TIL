# Grep

### ▼Grepとは
&ensp;・フォルダやファイルの種類を指定して一括して検索<br>
&ensp;・Grep結果をエクセルにまとめたりする場合など、基本的にはテキストエディタ（秀丸等）でGrepするのがおすすめ<br>
